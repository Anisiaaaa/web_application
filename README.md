# Симулятор Центрального банка

## Описание:
Проект  является игровым симулятором, моделирующим деятельность Центрального банка. Пользователь играет за Центральный банк: управляет основными инструментами денежно-кредитной политики — ключевой ставкой, нормой резервирования, объемом денежной массы и другими. Игра длиться несколько кварталов. Цель игрока — оптимально ответить на внешнее событие в каждом квартале, и в итоге способствовать стабильности банковской системы и сдержать инфляцию.
Для оценки последствий решений в симуляции отображается различные макроэкономические показатели, а также состояние ключевых агентов: домохозяйств, производителей, банков. Задача проекта — наглядно продемонстрировать принципы работы денежно-кредитной политики.
